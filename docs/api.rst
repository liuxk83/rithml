API Reference
=============

RithML contains four main modules, each corresponding to a category of machine learning algorithms. These categories and modules are listed below:

* Classification (:py:mod:`rithml.classification`)
* Regression (:py:mod:`rithml.regression`)
* Dimensionality reduction (:py:mod:`rithml.dimred`)
* Clustering (:py:mod:`rithml.clustering`)

Additionally, all classes in these modules inherit from the :py:class:`rithml.base.BaseModel` class, which implements the :py:func:`rithml.base.BaseModel.get_params` and :py:func:`rithml.base.BaseModel.set_params` methods.
